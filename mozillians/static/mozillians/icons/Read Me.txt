
This is a custom Mozillians icon set built on top of FontAwesome with icomoon.io

FILES
demo.html ~ a list of all the glyphs in the font along with their codes/ligatures.
style.css ~ include in the site in order to use the icon font
Mozillians.json ~ exported from Icomoon, re-import this to make changes and re-export so the changes stays in Git.

USAGE
The same as Fontawesome, but with a few extra classes.
